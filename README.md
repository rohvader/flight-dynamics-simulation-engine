\# Flight Dynamics Simulation Engine



A Unity/C# real-time simulation project focused on modular flight dynamics, runtime physics modelling and simulation architecture.



\## Repository Scope



This repository focuses on the source code, simulation architecture and modelling components behind the project. Large visual assets and third-party Unity assets are excluded from the public repository, so this is intended as a code and modelling portfolio rather than a fully packaged Unity build.



\## Technical Focus



\- Unity/C# simulation architecture

\- Modular flight dynamics scripts

\- Runtime physics modelling

\- Scene-based simulation setup

\- Python + MATLAB coefficient derivation pipeline for model calibration and validation



\## Repository Structure



\- `Assets/scripts/` — C# simulation and control scripts 

\- `Assets/scenes/` — Unity scene files

\- `Packages/` — Unity package dependencies

\- `ProjectSettings/` — Unity project configuration
\-`lift_drag_analysis/` - Data analysis files; CFD data extracted into .json files, regression and interpolation .py files
